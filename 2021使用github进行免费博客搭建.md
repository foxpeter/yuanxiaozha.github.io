# github进行免费博客搭建

### 1、准备工作

---

- 注册github账号，地址如下

```bash
https://github.com/join?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home
```

- 下载github客户端，方便之后操作github上的文件上传，地址如下

```bash
https://desktop.github.com
```

- markdown语法（可选）

### 2、整体流程介绍

---

1. 注册github
2. 在github创建新资源库
3. 设置博客，选择博客主题
4. 使用github客户端把博客文件下载到本地
5. 本地进行博客的编写，上传到github博客
6. 查看博客发布结果
